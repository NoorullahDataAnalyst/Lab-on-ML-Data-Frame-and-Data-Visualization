# Lab-on-ML-Data-Frame-and-Data-Visualization
## Global Stock Market Ranking & Analysis (2026) 🌍📈

## 📌 Project Overview
This project provides a comprehensive analysis of the World Wide Stock Market Rankings projected for 2026. By comparing Market Capitalization against National GDP, we identify which countries lead the global financial stage and use Machine Learning to understand the correlation between a nation's economic output and its stock market value.
The project is condensed into a highly efficient 10-cell workflow, moving from data ingestion to predictive modeling.
## 🚀 Key Features

* Data Wrangling: Uses pandas to manage global financial metrics.
* Buffett Indicator: Implementation of Market Cap-to-GDP ratios to identify market valuation levels.
* Visual Analytics:
* Horizontal bar charts for global rankings.
   * Scatter plots for economic correlation.
   * Regression line overlays for ML visualization.
* Predictive Modeling: A Linear Regression model to estimate Market Cap based on GDP trends.

## 🛠️ Tech Stack

* Language: Python
* Libraries:
* Pandas (Data manipulation)
   * Matplotlib & Seaborn (Visualizations)
   * Scikit-Learn (Machine Learning)
   * NumPy (Numerical operations)
